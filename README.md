# 📊 Sales Dashboard Design using Power BI

## 📌 Project Overview

This project was developed as part of a Data Analytics Internship. The objective is to design an interactive business dashboard using **Power BI** to help stakeholders analyze sales performance, profitability, and business growth.

---

## 🎯 Objective

The dashboard provides valuable business insights by visualizing:

- Total Sales
- Total Profit
- Total Orders
- Total Quantity Sold
- Sales Trend Over Time
- Regional Performance
- Category-wise Profit
- Customer Segment Analysis

---

## 🛠️ Tools Used

- Microsoft Power BI Desktop
- Microsoft Excel / CSV Dataset
- GitHub

---

## 📂 Dataset

**Dataset:** Superstore Sales Dataset (Kaggle)

The dataset includes:

- Order Date
- Sales
- Profit
- Quantity
- Discount
- Category
- Sub-Category
- Region
- Customer Segment
- Product Name

---

## 📈 Dashboard Features

- 📌 KPI Cards
  - Total Sales
  - Total Profit
  - Total Orders
  - Total Quantity

- 📈 Sales Trend Analysis

- 📊 Profit by Category

- 🌍 Sales by Region

- 🍩 Sales by Customer Segment

- 🏆 Top 10 Products by Sales

- 🎛 Interactive Filters (Slicers)
  - Region
  - Category
  - Segment
  - Order Date

---

## 📊 Key Insights

- Technology category generated the highest sales.
- Consumer segment contributed the largest share of revenue.
- Sales trends help identify seasonal patterns.
- Regional analysis highlights high-performing markets.
- Interactive filters enable detailed business analysis.

1. Download or clone this repository.
2. Open `Dashboard.pbix` using Microsoft Power BI Desktop.
3. Load the dataset if prompted.
4. Use the slicers to filter the dashboard.
5. Explore the interactive visualizations and insights.
